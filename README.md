# 数独解决方案
该解决方案包含以下几个项目
>SodukuBase 数独基础类库  
>SodukuBuilder 数独生成类库  
>SodukuQuestion 数独出题类库  
>SodukuSolver 数独求解类库  
>SodukuWinformUi WinformUI类库  
>SodukuTest 数独测试类库  

## 数独基础类库

设置盘面每个位置的信息

## 数独生成类库
生成完整数独。

## 数独出题类库
确保生成的数独题目是标准数独,有且仅有唯一解。

## 数独求解类库
采取非递归的方式求解数独，并记录数独求解过程


## WinformUi类库
以winform的形式展示数据，并构造求解过程
提供数独题目,并辅助玩家求解。

## 数独测试类库
单元测试类库
